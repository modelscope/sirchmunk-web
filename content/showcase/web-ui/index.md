Sirchmunk's Web UI provides a unified interface for real-time chat with streaming search logs, source citations, knowledge cluster browsing, and system monitoring — all with dark/light theme support and bilingual localization.

## Sirchmunk Web UI in Action

The demo below showcases Sirchmunk's Web UI — a full-featured chat interface for intelligent document search. Users can ask natural-language questions and receive real-time streaming answers with inline search logs, source citations, and evidence highlights. The UI also provides knowledge cluster analytics and system monitoring dashboards.

![Sirchmunk Web UI demo](Sirchmunk_Web.gif "Sirchmunk Web UI with streaming chat, source citations, and knowledge analytics")

**Features:**
- Streaming chat with inline search logs
- File-based RAG and web-augmented search modes
- Knowledge cluster analytics
- System health and LLM usage monitoring
- Dark/light theme with English and Chinese localization
- Natural-language queries with real-time evidence and citations
