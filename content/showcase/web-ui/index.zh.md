Sirchmunk 的 Web UI 提供了统一界面，支持实时流式聊天（含搜索日志）、来源引用、知识簇浏览和系统监控 — 全部支持深色/浅色主题和中英双语。

## Sirchmunk Web UI 演示

下方演示展示了 Sirchmunk 的 Web UI — 一个功能完善的智能文档搜索聊天界面。用户可以使用自然语言提问，并实时获得流式回答，同时附带内嵌搜索日志、来源引用和证据高亮。界面还提供知识簇分析和系统监控仪表盘。

![Sirchmunk Web UI 演示](Sirchmunk_Web.gif "Sirchmunk Web UI 流式聊天、来源引用和知识分析")

**功能特性：**
- 流式聊天，内嵌搜索日志
- 基于文件的 RAG 和网络增强搜索模式
- 知识簇分析
- 系统健康和 LLM 使用量监控
- 深色/浅色主题，中英文本地化
- 自然语言查询，实时证据与引用
